3. 	
--soft - отменит коммит, но сохранит изменения в файлах
--hard - отменит коммит и удалит изменения, сделанные коммитом.
git reset --soft HEAD~2 - отменит 2 последних коммита, 
сохранив изменения. Будут видны в git status.
