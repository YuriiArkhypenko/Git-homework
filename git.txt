3. Как получить на локальный компьютер изменения с github.com

 git fetch для получения новых работ, выполненных другим програмистом
 merge используется для слияния работы
 git pull — это тоже самое что  git fetch и git merge в одной команде


Дополнительно 

Какую информацию нам дают команда :
git status — показывает состояние изменений
git log - показывает историю коммитов в обратном порядке
git remote -v просмотр удаленных репозиториев Git (-v включает URL-адрес каждого подключения)
